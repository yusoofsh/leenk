# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, report it privately by emailing [me@yusoofsh.id](mailto:me@yusoofsh.id). Include reproduction steps, affected routes or versions, impact, and any supporting evidence.

Please do not open a public issue before the report has been reviewed. You should receive an acknowledgement within seven days and a status update after the initial triage.
